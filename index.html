Minha loja de bonsai

O projeto é uma página web criada com HTML e CSS. O objetivo é criar um ambiente virtual onde seja possível comprar bonsais para que o clinte consiga comprar bonsais de alta qualidade.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho como o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de bonsais em uma seção flexível (.bonsais). Cada bonsai é destacado em seu próprio contêiner (.bonsai), mostrando uma imagem do bonsai. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de bonsais.

Flashcards

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, que mostra a pergunta e quanto clica na pergunta mostra a resposta, o assunto dos Flashcards é skate. O objetivo é que as pessoas conheçam melhor o mundo do skate e assim acabe que mais pessoas se interessem pelo esporte.

o projeto foi feito utilizando o html parte estrutural do site. o JavaScript foi utilizado na parte das perguntas e das respostas, e também é ele que faz o flashcard virar. Já o css foi utilizado na decoração do texto para deixar um site mais harmônica visualmente.
